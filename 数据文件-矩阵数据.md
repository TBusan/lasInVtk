## **数据文件-矩阵数据**

文件名: 测线_通道号.data

```
道数量 * 采样点 * bit数量 的字节文件
```



## **道坐标文件**

文件名: 测线_通道号.cor

```
第一列：道序号 样例：18

第二列：日期  样例：2025-03-16	

第三列：时间  样例：06:00:13:000

第四列：北坐标 32.2467083333	

第五列：坐标类型 样例：N	

第六列：东坐标 118.9311076667	

第七列：坐标类型 样例：E

第八列：高程值 单位：M 样例：23.50	

第九列：高程 样例：M 

第十列：解状态
```
